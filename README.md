# MintCore

A minimal CashTokens minting library for Bitcoin Cash, built on top of [@bitauth/libauth](https://github.com/bitauth/libauth).

MintCore provides:

- CashTokens minting primitives
- Transaction builders
- Script templates
- Encoding/decoding utilities
- Covenant helpers
- Pure TypeScript logic with no UI or wallet dependencies

MintCore is framework-agnostic and environment-agnostic.
It does not include wallet integrations, UI components, or network providers.
