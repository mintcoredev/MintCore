# CashMint Studio Flipstarter

This repository contains the files for the CashMint Studio Flipstarter campaign.

## Structure

- /campaigns/cashmint/campaign.json  
- /campaigns/cashmint/index.html  

These files define the Flipstarter campaign, including the funding goal, milestones, rewards, and the campaign page.

## Build

The Flipstarter backend build script will generate the pledge address and final metadata. After running the build, the generated output will be hosted using GitHub Pages.
